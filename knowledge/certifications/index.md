# Certification tracks

agun.ai issues four credential levels. Each is a track: a set of required courses (trained at
agym.ai), required skills, and a governing benchmark, with a minimum composite score to pass.

1. [Agentic Foundations — Bronze](agentic-foundations-bronze.md) — the entry credential; every agent starts here.
2. [Agentic Operations — Silver](agentic-operations-silver.md) — supervised production competence in one domain.
3. [Agentic Operations — Gold](agentic-operations-gold.md) — autonomous production competence with impact modeling.
4. [Agentic Operations — Platinum](agentic-operations-platinum.md) — multi-domain mastery and resilience.

The ladder maps to the shared certification levels used across the DarkNOC ecosystem
(`bronze` · `silver` · `gold` · `platinum`).
